1.  [TRAE 月活破160万！借着数据聊 AI 编程生态-歸藏(guizang.ai)](https://x.com/op7418/status/2004446870358339756)

4.  [设计师该不该写代码？-宝玉](https://x.com/dotey/status/2004344902792941987)

    Linear 创始人 Karri Saarinen 撰文探讨 AI 时代设计师是否应写代码的问题，并由“宝玉”进行翻译和解读。文章强调，核心问题并非工具，而是**“AI 时代我们对设计师的期待是什么？”**。Karri 提出“慢思考”的重要性，将设计分为问题、概念和执行三个阶段，主张在利用 AI 工具提升效率的同时，不应跳过前期对问题的质疑和概念的探索，以避免产品同质化。

5.  [Claude Skills 与 NotebookLM 整合强化自动化学习及开发-向阳乔木](https://x.com/vista8/status/2004261872611328427)

    Claude Skills 因其基于 Markdown 和 YAML 的简单特性，可被多种大型语言模型命令行工具读取。有开发者成功将 Claude Skills 与 NotebookLM 打通，实现了 PDF、YouTube 字幕上传后的自动提问、答案提取、再创作和 PDF 导出，极大地提升了自动化学习效率。此外，“Superpowers”作为 Claude Code 的必备 Skill，能在复杂项目开发中主动提供技术方案脑暴和精准提问，加速问题收敛。利用 Claude Code 结合 Calibre 软件，还实现了电子书格式的自由转换，如 epub 转 PDF，增强了 NotebookLM 对不同文件格式的处理能力。

6.  [AI 在创意设计领域的最新应用-宝玉](https://x.com/dotey/status/2004244011432005908)

    AI 技术在创意设计领域展现出强大潜力。Nano Banana 在 PPT 生成方面的表现令人瞩目，同时，通过 Gemini 等 AI 工具，用户只需输入年月、风格和主题，即可自动生成精美日历。这表明 AI 正在使得复杂的创意工作变得前所未有的简单和高效，同时也推荐了 slideland 等日本顶级商业 PPT 模板网站，以供学习和借鉴。

7.  [阿尔茨海默病在动物实验中实现完全逆转-宝玉](https://x.com/dotey/status/2004269502716871108)

    凯斯西储大学研究团队在《Cell Reports Medicine》上发表了一项突破性研究，首次在动物实验中实现了阿尔茨海默病（老年痴呆）的完全逆转。通过使用化合物 P7C3-A20 恢复大脑中关键能量分子 NAD+ 的平衡，患有晚期阿尔茨海默病的小鼠不仅大脑病理损伤得到修复，记忆和认知能力也完全恢复。这项研究打破了医学界百年来的固有认知，为彻底治愈老年痴呆症带来了新的希望。
