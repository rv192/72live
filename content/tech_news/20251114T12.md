1.  [ChatGPT 上线群聊功能 - 小互](https://x.com/imxiaohu/status/1989163098146771436)

    ChatGPT 已正式上线群聊功能，支持最多20人共同与AI互动。该功能具备智能判断发言时机、被 @ 提及时强制响应、支持 emoji 表情以及在图片生成中引用用户头像加入个性化元素等特性。目前，该功能正在日本、新西兰、韩国和台湾地区进行试点。群聊还提供了丰富的管理选项，包括自定义群名、成员管理、静音通知，以及为群聊单独设置**自定义指令（Custom Instructions）**，允许用户定义ChatGPT在群聊中的语气、角色或参与方式，例如使其扮演“会议记录助理”或“语气更轻松幽默”的角色。

2.  [免费语音转文字工具“代体”改名为“闪电说” - 向阳乔木](https://x.com/vista8/status/1989152571353944539)

    免费语音转文字工具“代体”已更名为“**闪电说**”，并对官网进行了重新设计。经测试，该工具采用的文本转写模型 **GLM4.6** 表现出色，几乎无延迟感。结合专业麦克风使用时，用户体验极佳，被视为进行“Vibe Coding”和撰写长文的利器，其性能甚至优于付费工具 Monologue 和 Whispr Flow。这一更新使得高质量的语音转文字服务更加普及和便捷。

3.  [教你如何在 Codex CLI 里面用 SKILLs - 宝玉](https://x.com/dotey/status/1989146187786494351)

    一篇教程介绍了如何在 **Codex CLI** 中高效利用 **SKILLs**。用户需在项目目录下创建“.claude/skills”目录，并将所需 skill 文件复制其中。通过手动 `@` 提及相应的 skill 文件，Agent 即可学习并掌握该 skill，例如用于创建 whiteboard 项目。此方法不仅适用于 Codex CLI，也兼容 TRAE、Cursor、GitHub Copilot 等主流编码代理，凸显了 SKILL 设计的**超前性和通用性**，为AI辅助编程提供了新的范式。

4.  [Amazon 被曝利用AI克隆音色引发配音演员维权 - 宝玉](https://x.com/dotey/status/1989139235593679071)

    Amazon 在广告制作中被曝出争议事件。在与配音演员穆雪婷合作的七集广告中，Amazon 在录制完前三集后，**利用AI技术克隆了穆雪婷的音色**，并在未经授权和支付报酬的情况下，使用AI完成后续四集的配音。此举引发了穆雪婷在微博上的公开维权，凸显了AI技术在知识产权和劳动者权益方面带来的法律和道德挑战，引发了业界对AI内容生成伦理的广泛讨论。

5.  [AI输入格式化：Markdown与XML的选择 - 宝玉](https://x.com/dotey/status/1989132223703060637)

    文章探讨了使用 Markdown 或 XML 对AI输入内容进行格式化的重要性，旨在帮助AI快速理解文本结构。对于需要通过样式突出重点的简单文本，**Markdown** 足够有效。然而，当输入内容复杂，包含多种数据源，且Markdown无法有效区分不同部分时（例如处理多篇文章），**XML** 能够提供更清晰的结构化表示，从而提高AI处理的效率和准确性。选择合适的格式化方式是优化AI交互的关键。

6.  [Google DeepMind展示SIMA 2在Genie 3世界中的能力 - Demis Hassabis](https://x.com/demishassabis/status/198912494443334728)

    Google DeepMind 负责人 Demis Hassabis 宣布了 **SIMA 2** 在 **Genie 3** 创建的模拟3D世界中的惊人进展。Genie 3 是DeepMind的世界模型，能够生成多样化的可交互3D环境，而SIMA 2是一个通用的AI代理，能够遵循自然语言指令。此次测试展示了SIMA 2在复杂虚拟环境中**理解和执行任务**的强大能力，预示着AI在游戏、虚拟现实和模拟训练等领域的潜在应用将迎来突破。

7.  [NotebookLM推出自定义视频概览创建功能 - Demis Hassabis](https://x.com/demishassabis/status/1989124321445572743)

    NotebookLM 正式推出了一项新功能，允许用户创建**自定义视频概览**（custom video overview）。这项更新极大地增强了用户利用AI工具生成视频摘要和回顾的能力，从而提升了信息消化和内容创作的效率。该功能将使用户能够快速掌握长篇内容的精髓，并以视觉化的形式呈现，预计将在教育、新闻媒体和研究等领域提供显著便利。
