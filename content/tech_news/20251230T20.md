### 过去4小时全球AI要闻

1.  [Meta 收购AI代理公司 Manus，估值数十亿美元 - Orange AI](https://x.com/oran_ge/status/2005769693790130481)
    Meta 宣布成功收购人工智能代理公司 Manus，此次交易价格据称高达数十亿美元，此前 Manus 正寻求以20亿美元估值进行新一轮融资。此次收购谈判在极短时间内完成，仅历时十余天，彰显了AI领域 M&A 的快速节奏和巨头对通用AI代理的激烈争夺。此次收购被业界视为具有里程碑意义的事件，预计将对中国的AI投资和创业生态产生深远影响。

2.  [Novita Labs 在 Hugging Face 上的月请求量突破1000万 - clem 🤗](https://x.com/ClementDelangue/status/2005671337415569791)
    AI推理服务提供商 Novita Labs 宣布在 Hugging Face 平台上取得了显著成就，其每月请求量已成功突破1000万大关。这一里程碑标志着 Novita Labs 在AI推理服务领域的强劲增长和技术实力，进一步巩固了其作为重要AI基础设施提供商的地位。

3.  [吴恩达 (Andrew Ng) 分享AI系统构建技能培养建议 - Andrew Ng](https://x.com/AndrewYNg/status/2005702832524255475)
    人工智能领域领军人物吴恩达强调，在AI技术飞速发展的时代，开发者应通过**学习AI课程**、**实践构建AI系统**以及**阅读研究论文**来提升自身技能。他指出，直接投入实践而不理解基础是无效的，可能导致重复造轮子。理论学习与动手实践的结合至关重要，同时鼓励开发者阅读研究论文以获取前沿知识和灵感。

4.  [Claude Code 可视化工作流编辑器发布，支持自然语言修改 - 小互](https://x.com/imxiaohu/status/2005794552142258439)
    Claude Code 推出了一款可视化工作流编辑器，允许用户通过拖拽节点和自然语言交互来创建和修改自动化工作流。这款工具降低了AI开发门槛，使非编程人员也能轻松设计如文档总结、代码分析或网页数据抓取等自动化流程。用户可以在 VSCode 中通过画布界面构建复杂的工作流，并用自然语言指令进行调整。

5.  [免费电子书《算法与机器学习导论》发布 - 宝玉](https://x.com/dotey/status/2005749917222830081)
    Justin Skycak 为美国高中数学/CS课程 Eurisko 编写的免费电子书《算法与机器学习导论》已开放下载。该教材旨在让高中生从零开始，使用 Python 手动实现所有算法，甚至复现AI学术论文。内容涵盖从二进制、排序算法到K-Means聚类、神经网络反向传播等，终点达到研究级水平，特别适合自学者和教育者深入理解ML原理。

6.  [GLM 4.7 成为 Baseten 团队默认编码模型 - sarah guo](https://x.com/saranormous/status/2005755682189136156)
    据 @amiruci 分享，在 Baseten 团队内部，**GLM 4.7** 已超越基准测试，成为日常编码的首选模型。这表明 GLM 4.7 在实际开发中展现出卓越的实用性和效率，赢得了开发者的高度认可。

7.  [Hugging Face 和 Pollen Robotics 合作Reachy Mini机器人，集成Grok语音API - clem 🤗](https://x.com/ClementDelangue/status/2005709977982583182)
    Hugging Face 与 Pollen Robotics 合作推出了 Reachy Mini 机器人，用户可以在圣诞节期间享受动手组装的乐趣。该机器人还能结合 Grok 语音 API 进行编程，为机器人技术和语音AI的结合提供了有趣的实践平台，暗示着未来机器人开发的易用性和交互性将大大增强。

8.  [Replit 倡导“Vibe Coding”概念，培养“1000倍工程师” - Amjad Masad](https://x.com/amasad/status/2005816049971679741)
    Replit 首席执行官 Amjad Masad 推广“Vibe Coding”理念，旨在让没有编码经验的用户也能轻松构建应用。他指出，通过直观的AI辅助工具，任何人都能快速入门并掌握编程技能。Replit 与 Campusgrad 团队合作推出了 "Vibe Coding 101" 课程，以赋能更多人成为能够利用AI大幅提升效率的“1000倍工程师”。

9.  [Claude Pro/Max 用户本地开发Agent SDK无需API Key - 宝玉](https://x.com/dotey/status/2005714387060023749)
    对于订阅了 Claude Pro/Max 的用户，现在进行本地 Claude Agent SDK 开发时，无需再使用传统的 API Key。用户可以直接共享其 Claude 账号进行登录和开发，极大地简化了开发流程和认证步骤，提升了开发便利性。

10. [SWE Agent ACI设计原则指导LLM技能开发 - 宝玉](https://x.com/dotey/status/2005713667850133963)
    @dongxi_nlp 指出，在开发基于大型语言模型（LLM）的“Skill”时，应遵循 SWE Agent 的 ACI 设计原则，即 **Actions should be Simple (简单)**、**Actions should be Compact (紧凑)** 和 **Actions should be Interpretable (可解释)**。强调 LLM 更倾向于使用基于文件系统和 Bash 的工具，而非高度自定义的复杂工具，这为设计高效的LLM代理提供了重要指导。

11. [智谱AI（Zhipu AI）即将港股上市，成为“全球大模型第一股” - Orange AI](https://x.com/oran_ge/status/2005801019742248991)
    北京智谱华章科技股份有限公司（智谱）宣布启动招股，预计将于2026年1月8日正式在港交所主板挂牌上市，股票代码为“2513”。这将使智谱成为**全球首家大模型上市企业**，标志着中国AI企业在全球资本市场上的重大突破，被誉为“中国骄傲”。
