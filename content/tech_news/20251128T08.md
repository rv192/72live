8.  [Google AI负责人Jeff Dean揭示AI指数级进步的底层逻辑 - 向阳乔木](https://x.com/vista8/status/1993868088950092123)

    Google AI 负责人 Jeff Dean 在斯坦福的分享中指出，当前AI的强大是过去十五年**一系列技术突破叠加**的产物，而非单一突破。他回顾了AI发展的关键节点：2012年 Google Brain **无监督学习**让模型学会识别“猫”；Word2Vec 技术通过**高维向量**使机器理解语言语义；2015年 **TPU专用硬件**的推出解决了算力瓶颈，将计算速度提升数十倍甚至数千倍；2017年 **Transformer架构**彻底改变了语言处理，成为所有大模型的基础；以及稀疏模型、蒸馏和**思维链（Chain-of-Thought）**等训练技巧显著提升了模型效率与推理能力。Dean强调，AI的进步速度是**指数级的**，从2022年能做初中数学题到2024年Gemini 2.5 Pro达到国际奥数金牌水平，仅用了两年时间。他预测AI将对医疗、教育、科研产生巨大影响，并呼吁在最大化益处的同时，正视并最小化如错误信息传播等潜在弊端。

10. [Anthropic 探索长周期 AI Agent 的有效运行框架 - 宝玉](https://x.com/dotey/status/1993904856067694612)

    Anthropic 在其博客中深入探讨了如何解决 AI Agent 在长时间、复杂任务中面临的“记忆”和上下文窗口限制。文章分析了 Agent 常出现的三种失败模式：**一次性处理过多任务、过早宣布完成以及敷衍测试**。针对这些问题，Anthropic 提出了借鉴人类软件工程实践的解决方案，包括引入类似团队分工的协作机制，将复杂任务分解为可追踪验证的子任务，建立清晰的交接机制，并将“记忆”外化为文件（如 JSON 格式的功能清单和 Git 历史）以便 Agent 持续推进任务。特别强调通过配备 **Puppeteer 等浏览器自动化工具**，使 Agent 能够像真人用户一样进行端到端测试，大幅提升功能验证的准确率。
